Deployment with Docker

Prerequisites
-------------

* Docker 17.05+.
* Docker Compose 1.17+

Configuration with enviroment vars
----------------------------------

To setup the server for your personal needs, you need to create 2 files.
``.envs/.production/.django`` & ``.envs/.production/.postgres``.
